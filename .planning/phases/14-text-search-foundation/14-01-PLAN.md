minsearch library API (from Phase 13 installed):
```python
import minsearch

# Create index with field boosting
index = minsearch.Index(
    text_fields=["content", "title"],  # Fields to index for text search
    keyword_fields=["filename", "chunk_id"],  # Fields for exact match filtering
    boost_config={"title": 2.0, "content": 1.0}  # Field weights
)

# Index documents (list of dicts)
index.fit(documents)

# Search with top-K
results = index.search(
    query="search query",
    num_results=5  # top-K parameter
)
# Returns: list of dicts with original fields + 'score' field
```
